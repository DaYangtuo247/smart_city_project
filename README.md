## 技术栈

该项目主要使用`Vue`搭配`Echarts`采用组件化的方式开发完成,此外项目中陆续使用有

- Vue3

- WebSocket

- axios

- webpack

- ES6+

- Tailwind

- gaode jsapi/loca

- mitt

  ​    

## 如何运行该项目？

```bash
# 首先你应当克隆该项目
git clone https://github.com/DaYangtuo247/smart_city_project
```

环境要求：node v18.16.0 + @vue/cli 5.0.8

### 启动前端

```bash
cd view
npm install
npm run smart-city
```

### 启动后端

```bash
cd server
npm install
node app.js
```

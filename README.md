## 如何运行该项目？

```bash
# 首先你应当克隆该项目
git clone https://github.com/DaYangtuo247/smart_city_project
```

环境要求：node v18.16.0 + @vue/cli 5.0.8

### 启动前端

```bash
cd smart-city
npm install //只需第一次运行时使用
npm run smart-city
```

### 启动后端

```bash
cd server
npm install //只需第一次运行时使用
node app.js
```

### 启动ai

```bash
cd chatgpt
npm install //只需第一次运行时使用
npm run ai
```

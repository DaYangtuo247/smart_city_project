## 技术栈

该项目主要使用`Vue`搭配`Echarts`采用组件化的方式开发完成,此外项目中陆续使用有

- VueRouter

- Vue

- WebSocket

- axios

- webpack

- ES6+

- less

- Tailwind

- gaode jsapi/loca

    

## 如何运行该项目？

```bash
# 首先你应当克隆该项目
git clone https://github.com/DaYangtuo247/smart_city_project
```

### 启动前端工程

```bash
cd view
npm install
npm run serve
```

### 启动后端工程

```bash
cd server
npm install
node app.js
```

### 启动ai

 ```bash
 cd chatgpt
 npm install 
 npm run ai
 ```
